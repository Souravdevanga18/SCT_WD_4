# Smart Task Manager Web App

This project was developed as part of the SkillCraft Technology Web Development Internship.

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript

## ✨ Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Task Search Functionality
- Completed & Pending Filters
- Priority Labels
- Notes Section
- Dashboard Analytics
- Task Completion Percentage
- Dark / Light Theme
- Local Storage Support
- Responsive Design
- Animated Modern UI

## 📊 Dashboard Features

- Total Tasks Counter
- Completed Tasks Counter
- Dynamic Progress Percentage

## 🎯 Objective

To build an advanced and responsive task management web application using JavaScript, local storage, and modern frontend UI design techniques.

- index.html
- style.css
- script.js
